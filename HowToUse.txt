-English Tutorial
1. Download Python 3.9.x (Video by Amit Thinks : https://youtu.be/uDbDIhR76H4)
2. Download Modules (run install.bat)
3. Edit Config file(config.txt)
You can get your mouse position with getpos.py
4. Execute(Run) runeng.bat as administrator (if you execute eng.py or execute w/out administrator
macro won`t work)
eng.py is English version of SeedReroller

-한국어 튜토리얼
1. 파이썬 3.9.x 를 다운받습니다 (Amit Thinks의 영상 참고 : https://youtu.be/uDbDIhR76H4)
2. 모듈들을 다운받습니다 (install.bat을 실행시키면 됩니다)
3. 설정 파일(config.txt)를 수정합니다
getpos.py 를 통해서 현재 마우스 위치를 확인할수 있습니다
4. runkor.bat 을 관리자 권한으로 실행합니다 (kor.py를 실행 또는 관리자 권한으로 실행을 하지
않을 시 매크로가 작동하지 않습니다)
kor.py는 한국어 버젼의 SeedReroller 입니다
