-English Tutorial (No English ver for v1.1 now)
1. Download Python 3.9.x (You can install Anaconda)
2. Download Modules (run install.bat)
3. Edit Config file(config.txt)
You can get your mouse position with getpos.py
4. Open cmd as administrator and run eng.py with command (read github)
eng.py is English version of SeedReroller

-한국어 튜토리얼
1. 파이썬 3.9.x 를 다운받습니다 (아나콘다도 상관없음)
2. 모듈들을 다운받습니다 (install.bat을 실행시키면 됩니다)
3. 설정 파일(config.txt)를 수정합니다
getpos.py 를 통해서 현재 마우스 위치를 확인할수 있습니다
4. cmd를 관리자 권한으로 실행한 뒤 명령어로 kor.py를 실행합니다 (깃허브를 읽어보세요)
kor.py는 한국어 버젼의 SeedReroller 입니다
